# 小说写作系统 (Novel Writing System)

一套完整的小说创作方法论和写作工具系统，专注于网文/玄幻小说的开篇设计和创作指导。

## 📚 项目概述

本项目整合了网文写作的最佳实践、经典开篇结构分析和成功案例研究，为小说创作者提供系统化的创作方法论和实用工具。

## 🎯 核心功能

### 1. 黄金三章设计方法论

**黄金三章**是网文创作中最关键的三个章节，直接决定读者是否会继续阅读。

- **第一章**：3秒抓眼球 - 从"炸楼"开始写，让读者立刻紧张起来
- **第二章**：强化冲突 - 给主角一个紧迫目标 + 设置具体阻碍
- **第三章**：爽点反馈 - 让读者尝到甜头，即使只是小胜利

详细方法论请查看：[黄金篇章设计方法论](docs/research/golden-three-chapters-methodology.md)

### 2. 开篇设计检查清单

#### 第一章检查清单
- [ ] 前300字有爆点（危机/金手指/反转）
- [ ] 主角已登场
- [ ] 基础信息一句话交代
- [ ] 避免寡淡开场、信息轰炸、拖沓入题
- [ ] 视角聚焦主角（100%主角视角）

#### 第二章检查清单
- [ ] 有紧迫目标（时间限制）
- [ ] 有具体阻碍（明确且强烈）
- [ ] 世界观通过剧情展开
- [ ] 金手指/机遇足够吸引

#### 第三章检查清单
- [ ] 有小爽点（打脸/突破/展示）
- [ ] 爽点与前两章呼应
- [ ] 主角闪光点展现
- [ ] 结尾有强力钩子

### 3. 爽点设计方法论

**爽文的核心公式**：主角爽 = 读者爽

**爽点的四大核心原则**：
1. **密**：缩短爽点周期，密度越高越易出成绩
2. **杂**：避免单一套路，多方位交替呈现
3. **重收获**：物质/地位等实际收获是最直接的爽点
4. **强落差**：通过巨大反差制造爽感

### 4. 题材专项指导

#### 玄幻/仙侠类
- 废材开局 + 金手指觉醒 + 打脸逆袭
- 被退婚/被羞辱 → 系统激活 → 小比第一轮

#### 都市重生类
- 重生时刻 + 带着记忆 + 第一桶金
- 中年失败重生 → 彩票股票房产 → 打脸前世仇人

#### 职场爽文类
- 职场危机 + 反击能力 + 维权成功
- PIP通知 → 法律漏洞 → 打脸HR

## 🛠️ 使用方法

### 快速自查

使用黄金三章评分表对您的开篇进行评估：

| 章节 | 满分 | 实际得分 |
|------|------|----------|
| 第一章 | 40 | |
| 第二章 | 30 | |
| 第三章 | 30 | |
| **总计** | **100** | |

**评分等级**：
- **90-100分**：黄金三章完美，预计高留存
- **80-89分**：黄金三章优秀，预计良好留存
- **70-79分**：黄金三章合格，需要优化后发布
- **60-69分**：黄金三章勉强合格，建议大幅优化
- **<60分**：黄金三章不合格，建议重写

### 一句话检查标准

- 第一章结束，读者会不会想"接下来会发生什么"？
- 第二章结束，读者会不会替主角"捏把汗"？
- 第三章结束，读者会不会觉得"有点爽，想看下一章"？

如果三个答案都是"是"，你的黄金三章就合格了。

## 📊 成功案例

### 《重生从咸鱼赢起》（打不死的小萧）
- 两万订完本神作
- "被动升职"的反套路设计
- 完美契合职场人心理需求

### 《我在职场学斩神》（卦乾）
- 双重身份极限拉扯
- 白天唯唯诺诺，晚上监狱典狱长

- 主打职场清醒语录
- 金句直击职场人心声
- 建立强烈情感共鸣

## 🚧 避坑指南

### 三大开篇雷区

| 雷区 | 表现 | 示例 |
|------|------|------|
| **寡淡开场** | 平平无奇 | "林风是个普通上班族，每天朝九晚五..." |
| **信息轰炸** | 大量设定 | "在玄天大陆东南方，有个三千年历史的青云宗..." |
| **拖沓入题** | 迟迟不入正题 | 先写三段天气、两段起床洗漱... |

### 不要给读者出选择题

**核心原则**：把剧情设计成"判断对错题"，而非"选择题"。

让所有读者对主角的行为达成共识，而不是产生分歧。
